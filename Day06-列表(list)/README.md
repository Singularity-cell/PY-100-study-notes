## 📝 核心知识点

### 1. 创建列表
```python
# 空列表
empty_list = []

# 字符串列表
fruits = ["苹果", "香蕉", "橙子", "葡萄"]

# 数字列表
numbers = [1, 2, 3, 4, 5]

# 混合类型列表（不推荐，但可行）
mixed = ["苹果", 12, True, 3.14]
```

### 2. 访问列表元素
```python
fruits = ["苹果", "香蕉", "橙子", "葡萄"]

# 正向索引（从0开始）
print(fruits[0])   # "苹果"
print(fruits[1])   # "香蕉"
print(fruits[2])   # "橙子"

# 负向索引（从-1开始）
print(fruits[-1])  # "葡萄"（最后一个）
print(fruits[-2])  # "橙子"（倒数第二个）

# 切片操作（获取子列表）
print(fruits[1:3])   # ["香蕉", "橙子"]（索引1到2）
print(fruits[:2])    # ["苹果", "香蕉"]（从开始到索引1）
print(fruits[2:])    # ["橙子", "葡萄"]（从索引2到结束）
```

### 3. 修改列表
```python
fruits = ["苹果", "香蕉", "橙子"]

# 修改元素
fruits[1] = "草莓"  # 列表变为 ["苹果", "草莓", "橙子"]

# 添加元素
fruits.append("葡萄")      # 末尾添加
fruits.insert(1, "芒果")   # 在索引1处插入

# 删除元素
del fruits[0]           # 删除索引0的元素
removed = fruits.pop()  # 删除并返回最后一个元素
fruits.remove("草莓")   # 删除第一个匹配的元素
```

### 4. 常用列表方法
| 方法 | 作用 | 示例 |
|------|------|------|
| `append(x)` | 在末尾添加元素 | `list.append(5)` |
| `insert(i, x)` | 在索引i处插入x | `list.insert(0, "头")` |
| `remove(x)` | 删除第一个值为x的元素 | `list.remove("苹果")` |
| `pop([i])` | 删除并返回索引i的元素 | `list.pop()` |
| `sort()` | 排序（升序） | `list.sort()` |
| `reverse()` | 反转列表 | `list.reverse()` |
| `index(x)` | 返回x的索引 | `list.index("香蕉")` |
| `count(x)` | 统计x出现的次数 | `list.count(5)` |
| `len(list)` | 获取列表长度 | `len(list)` |

### 5. 列表与for循环（重点！）
```python
# 遍历列表元素
fruits = ["苹果", "香蕉", "橙子"]
for fruit in fruits:
    print(f"水果：{fruit}")

# 遍历列表索引和元素
for i, fruit in enumerate(fruits):
    print(f"第{i+1}个水果：{fruit}")

# 遍历数字列表进行计算
numbers = [1, 2, 3, 4, 5]
total = 0
for num in numbers:
    total += num
print(f"总和：{total}")
```

### 6. 列表推导式（高级技巧，了解即可）
```python
# 传统方式
squares = []
for i in range(10):
    squares.append(i**2)

# 列表推导式（更简洁）
squares = [i**2 for i in range(10)]
```

## 🔍 常见问题
**Q：索引从0开始还是1开始？**
A：Python中列表索引从**0开始**。第一个元素索引是0，第二个是1，以此类推。

**Q：列表和字符串有什么区别？**
A：字符串是不可变的（不能修改单个字符），列表是可变的（可以修改元素）。

**Q：列表可以嵌套吗？**
A：可以！列表中可以包含其他列表，形成二维列表（如矩阵）。
