# Day04 - 条件语句（if-elif-else）学习指南

## 🎯 学习目标
1. 理解条件语句的概念和作用
2. 掌握 `if`、`elif`、`else` 的基本语法
3. 能够根据条件执行不同的代码块
4. 学会嵌套条件语句
5. 实践条件语句在实际问题中的应用

## 📚 核心知识点

### 1. 条件语句的基本结构
```python
if 条件:
    # 条件为真时执行的代码
else:
    # 条件为假时执行的代码
```

### 2. 多条件判断
```python
if 条件1:
    # 条件1为真时执行
elif 条件2:
    # 条件2为真时执行
else:
    # 所有条件都为假时执行
```

### 3. 条件表达式
- 比较运算：`>`, `<`, `>=`, `<=`, `==`, `!=`
- 逻辑运算：`and`, `or`, `not`
- 成员运算：`in`, `not in`

### 4. 嵌套条件语句
```python
if 条件1:
    if 条件2:
        # 条件1和条件2都为真时执行
```

## 🛠️ 实践项目：智能成绩评定系统

### 项目简介
创建一个成绩评定系统，根据输入的成绩自动给出等级评价。

### 功能需求
1. 接收用户输入的成绩（0-100分）
2. 根据成绩给出等级：
   - 90-100: 优秀
   - 80-89: 良好
   - 70-79: 中等
   - 60-69: 及格
   - 0-59: 不及格
3. 处理无效输入（如负数、大于100的数）

### 项目步骤

#### 步骤1：基础成绩评定
实现最基本的成绩等级判断，只使用 `if-else`。

#### 步骤2：多等级评定
使用 `if-elif-else` 实现完整的5个等级判断。

#### 步骤3：输入验证
添加输入验证，确保成绩在合理范围内（0-100）。

#### 步骤4：扩展功能
添加更多评价维度（如出勤率、作业完成情况）。

#### 步骤5：用户交互优化
优化用户界面，提供更好的交互体验。

## 🧪 代码示例（仅参考）

```python
# 示例1：基础if-else
age = 18
if age >= 18:
    print("已成年")
else:
    print("未成年")

# 示例2：多条件判断
score = 85
if score >= 90:
    print("优秀")
elif score >= 80:
    print("良好")
elif score >= 70:
    print("中等")
elif score >= 60:
    print("及格")
else:
    print("不及格")
```
